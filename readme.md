# Glint

**Glint** is a command-line note-taking engine, written in Go 1.24 by Stephen Malone.

```text
{
    "alpha": {
        "1000000": {
            "name": "alpha",
            "body": "Alpha (old).\n",
            "hash": "08734b08b7v207b2...",
            "init": "1000000",
        },
    },
}
```
